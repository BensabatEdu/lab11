# Lab11 - Programação Concorrente

Este repositório contém dois arquivos `.java` desenvolvidos para o laboratório 11 da disciplina de Programação Concorrente.

## Como Compilar

Para compilar os arquivos, utilize o `javac` no terminal. Execute os seguintes comandos:

```bash
javac MyPool.java
```

```bash
javac FutureHello.java
```

## Como Executar

Após a compilação, execute os programas com os comandos abaixo:

### Executar `MyPool.java`

```bash
java MyPool
```

### Executar `FutureHello.java`

```bash
java FutureHello
```
